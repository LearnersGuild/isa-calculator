## Getting Started

1. Fork and clone the repository.

2. Install and run:

    ```bash
    npm install
    npm run dev
    ```

3. If you change the `"reactToolbox"` section of `package.json`, you'll have to run the `toolbox` script.

    ```bash
    npm run build:theme
    ```
